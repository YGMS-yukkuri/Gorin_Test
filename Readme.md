# 技能五輪のB・C問題を解いていくやつ
---
### 2025/12/02
#### B1
グリッドの扱い方はもとより知っていたが、テンプレートを作成する際の`repeat([回数], [サイズ])`の記法を知らなかった。また、`grr --> grid-rows`や`grc --> grid-columns`といったVSCodeで使用可能な省略記法について気付きを得られた。
#### B2
`@keyframes`を用いたCSSのみでのアニメーション作成は行ったことがなかったため、宣言方法や作り方を知った。
#### B3
`input`要素を`display:none`指定して非表示にし、`label`要素としての当たり判定を確保してチェックボックスをオンオフできるようにし、それに応じたCSSを実装するという手順が新しい考え方として得られた。また、中にある丸い部分については、あまり難しく考えずに`margin`プロパティを用いて実装することで簡単に実装できる。

また、チェックされているときにスタイルを変更するという処理は`input:checked+label`のように表記することで実装することができる。また、`label .circle`と表記することでlabelのさらに子要素についても変更させることができる。
